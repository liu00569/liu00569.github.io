Description List

  1. a: It's anchor tag, we use it to create a link on webpage.
  2. html: It's the most Important tag, anything we add to the page must be inside opening html and closing html tags.
  3. body: It's the main body of the document. All the content of the web page must be added inside the opening body and closing body tag.
  4. article: It's used for containing independent piece of content.
  5. section: It's group of related content on the page.
